# Line-Follower
Basic line follower for Mechatronics Systems Design Class

# BOM
  Motor DC Pololu 12V 10:1 HPCB
  Perfboard
  IR Sensors Pololu QRT-8
  Arduino Pro Micro
  Driver TB6612FNG
  Step Down
  Led
  Printed Wheels with nanotape as tyre
  CNHL LiPo 2S 450mAh
